* # PortfolioHUB | Nicole Cruz

## Sobre o Projeto

O **PortfolioHUB** é uma plataforma centralizada para exibir e gerenciar projetos e portfólios digitais de forma organizada e profissional. Este projeto demonstra competências práticas em versionamento de código (Git), publicação automatizada via GitHub Pages e, fundamentalmente, a aplicação de **Inteligência Artificial (Google Gemini)** como guia para o planejamento e a segurança da aplicação.

* **Instituição:** Centro Universitário de Brasília (CEUB)
* **Curso:** Ciência de Dados e Machine Learning
* **Disciplina:** Bootcamp 
* **Período:** 1º Semestre / 2026

---

## Planejamento e Apoio da IA (Google Gemini)

Conforme as diretrizes da entrega final, o desenvolvimento deste HUB foi orientado pelo Google Gemini para garantir a conformidade com as melhores práticas de mercado:

* **Plano de Implantação:** Elaboração de um roteiro detalhado para a publicação e manutenção do sistema em ambiente de produção.
* **Guia de Segurança:** Uso da IA para definir políticas de proteção de dados, integridade de código e gestão de usuários.

---

## Configuração Técnica e Segurança

Este repositório implementa práticas de gestão profissional para garantir a robustez da solução:

* **Integração GitHub:** Uso de funcionalidades nativas para armazenamento, controle de versão e compartilhamento de código.
* **Gestão de Usuários:** Configuração de controle de acesso e definição de permissões de colaboração.
* **Políticas de Segurança:** Implementação de regras de segurança no repositório para garantir a conformidade com as práticas recomendadas pelo Google Gemini.

---

## Arquitetura do Repositório

A estrutura de diretórios foi estabelecida para facilitar a navegação e a documentação modular:

* **01-ecossistema-projetos:** Centralização de soluções práticas e análises aplicadas ao setor público (como projetos de Emendas Parlamentares e Bolsa Família).
* **02-fundamentos-python:** Repositório de aprendizado em lógica de programação, algoritmos e estruturas de dados em Python.
* **03-recomendações-conquistas:** Mapeamento de feedbacks e recomendações de professores conectados a perfis profissionais.
* **04-interface-experiência:** Código-fonte (HTML/CSS) responsável pela interface web e usabilidade do portfólio.
